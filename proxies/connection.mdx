## Endpoints

| Protocol | Host | Port |
|----------|------|------|
| HTTP/HTTPS | `p1.turnoxy.com` | `1318` |

## URL Format

```
http://{username}-{params}:{password}@{host}:{port}
```

```
http://sub_xxx-country-US:pass@p1.turnoxy.com:1318
└─┬─┘ └──┬──┘ └────┬────┘ └┬─┘ └──────┬──────┘ └┬─┘
  │      │         │       │          │         │
protocol user   params   pass       host      port
```

## HTTP/HTTPS

```bash
# HTTP target
curl -x "http://sub_xxx:pass@p1.turnoxy.com:1318" http://example.com

# HTTPS target
curl -x "http://sub_xxx:pass@p1.turnoxy.com:1318" https://example.com
```

<Note>
Proxy URL uses `http://` for both HTTP and HTTPS targets. HTTPS uses CONNECT tunneling.
</Note>
