## 1. Get Your Credentials

After subscribing to a plan, you'll receive credentials in this format:

| Component | Example |
|-----------|---------|
| Username | `sub_k8mX2pL9` |
| Password | `Qw4nR7vB3xY9mK2p` |

You can find your credentials in the [Dashboard](https://turnoxy.com/dashboard).

## 2. Make Your First Request

<CodeGroup>
```bash cURL
curl -x "http://sub_k8mX2pL9:Qw4nR7vB3xY9mK2p@p1.turnoxy.com:1318" \
     https://api.ipify.org
```

```python Python
import requests

proxies = {
    "http": "http://sub_k8mX2pL9:Qw4nR7vB3xY9mK2p@p1.turnoxy.com:1318",
    "https": "http://sub_k8mX2pL9:Qw4nR7vB3xY9mK2p@p1.turnoxy.com:1318"
}

response = requests.get("https://api.ipify.org", proxies=proxies)
print(response.text)
```

```javascript Node.js
const axios = require('axios');

const response = await axios.get('https://api.ipify.org', {
  proxy: {
    host: 'p1.turnoxy.com',
    port: 1318,
    auth: {
      username: 'sub_k8mX2pL9',
      password: 'Qw4nR7vB3xY9mK2p'
    }
  }
});

console.log(response.data);
```
</CodeGroup>

## 3. Add Parameters

Customize your requests by adding parameters to the username:

```bash
# Target US IPs
curl -x "http://sub_k8mX2pL9-country-US:Qw4nR7vB3xY9mK2p@p1.turnoxy.com:1318" \
     https://api.ipify.org

# Target Germany IPs
curl -x "http://sub_k8mX2pL9-country-DE:Qw4nR7vB3xY9mK2p@p1.turnoxy.com:1318" \
     https://api.ipify.org

# Sticky session (same IP for 10 minutes)
curl -x "http://sub_k8mX2pL9-session-mysession-ttl-10:Qw4nR7vB3xY9mK2p@p1.turnoxy.com:1318" \
     https://api.ipify.org
```
