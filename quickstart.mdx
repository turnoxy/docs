## 2. Make Your First Request

<CodeGroup>
```bash cURL
curl -x "http://sub_k8mX2pL9:Qw4nR7vB3xY9mK2p@p1.turnoxy.com:1318" \
     https://api.ipify.org
```

```python Python
import requests

proxies = {
    "http": "http://sub_k8mX2pL9:Qw4nR7vB3xY9mK2p@p1.turnoxy.com:1318",
    "https": "http://sub_k8mX2pL9:Qw4nR7vB3xY9mK2p@p1.turnoxy.com:1318"
}

response = requests.get("https://api.ipify.org", proxies=proxies)
print(response.text)
```

```javascript Node.js

console.log(response.data);
```

## 3. Add Parameters

Append parameters to the username with `-` delimiter:

```bash
# Target US IPs
curl -x "http://sub_k8mX2pL9-country-US:Qw4nR7vB3xY9mK2p@p1.turnoxy.com:1318" \
     https://api.ipify.org

# Target Germany IPs
curl -x "http://sub_k8mX2pL9-country-DE:Qw4nR7vB3xY9mK2p@p1.turnoxy.com:1318" \
     https://api.ipify.org

# Sticky session (same IP for 10 minutes)
curl -x "http://sub_k8mX2pL9-session-mysession-ttl-10:Qw4nR7vB3xY9mK2p@p1.turnoxy.com:1318" \
     https://api.ipify.org
```
