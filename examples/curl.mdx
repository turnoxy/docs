## Basic Requests

### Simple Request

```bash
curl -x "http://sub_k8mX2pL9:Qw4nR7vB3xY9mK2p@proxy.turnoxy.com:1318" \
     https://api.ipify.org
```

### Check Your Exit IP

```bash
curl -x "http://sub_k8mX2pL9:Qw4nR7vB3xY9mK2p@proxy.turnoxy.com:1318" \
     https://httpbin.org/ip
```

## Session Control

### Sticky Session

```bash
# Same IP for all requests with this session ID
curl -x "http://sub_k8mX2pL9-session-mysession:Qw4nR7vB3xY9mK2p@proxy.turnoxy.com:1318" \
     https://api.ipify.org

# Run multiple times - should return the same IP
```

### Custom Session TTL

```bash
# Session expires after 10 minutes
curl -x "http://sub_k8mX2pL9-session-mysession-ttl-10:Qw4nR7vB3xY9mK2p@proxy.turnoxy.com:1318" \
     https://api.ipify.org
```

### Rotating IPs

```bash
# No session = new IP each request
curl -x "http://sub_k8mX2pL9-country-US:Qw4nR7vB3xY9mK2p@proxy.turnoxy.com:1318" \
     https://api.ipify.org

# Run multiple times - should return different IPs
```

## Complete Examples

### Full Configuration

```bash
curl -x "http://sub_k8mX2pL9-pool-dc-country-US-state-california-city-los_angeles-session-task01-ttl-15-timeout-60:Qw4nR7vB3xY9mK2p@proxy.turnoxy.com:1318" \
     https://httpbin.org/ip
```

### With Custom Headers

```bash
curl -x "http://sub_k8mX2pL9-country-US:Qw4nR7vB3xY9mK2p@proxy.turnoxy.com:1318" \
     -H "User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36" \
     -H "Accept-Language: en-US,en;q=0.9" \
     https://httpbin.org/headers
```

### POST Request

```bash
curl -x "http://sub_k8mX2pL9-country-US:Qw4nR7vB3xY9mK2p@proxy.turnoxy.com:1318" \
     -X POST \
     -H "Content-Type: application/json" \
     -d '{"key": "value"}' \
     https://httpbin.org/post
```

### Download File

```bash
curl -x "http://sub_k8mX2pL9-timeout-120:Qw4nR7vB3xY9mK2p@proxy.turnoxy.com:1318" \
     -o output.html \
     https://example.com
```
