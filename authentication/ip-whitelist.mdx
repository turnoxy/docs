---
title: "IP Whitelist"
description: "Authenticate without password using whitelisted IPs"
---

Pre-register your IP addresses to skip password authentication.

## How It Works

1. Add your IP to the whitelist in Dashboard
2. Make requests without password
3. Only username (with parameters) required

## Configuration

| Setting | Value |
|---------|-------|
| Binding Level | Per Subscription |
| Max IPs | 3 per subscription |
| Format | IPv4 address |

## Usage

From a whitelisted IP, omit the password:

```bash
# IP whitelist
curl -x "http://sub_k8mX2pL9-country-US@p1.turnoxy.com:1318" \
     https://example.com

# Password auth (for comparison)
curl -x "http://sub_k8mX2pL9-country-US:Qw4nR7vB3xY9mK2p@p1.turnoxy.com:1318" \
     https://example.com
```

## Use Cases

- **Static IP servers** - No credentials in config files
- **Automation** - Simpler configuration
- **CI/CD pipelines** - Whitelist runner IPs

## Limitations

- IPv4 only
- Single IPs only (no CIDR)
- Max 3 IPs per subscription
